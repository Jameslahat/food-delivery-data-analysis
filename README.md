
# Food Delivery Data Analysis – Hackathon

##  Project Overview
This project analyzes food delivery transactional data by integrating CSV, JSON, and SQL datasets to extract meaningful business insights.

##  Datasets Used
- orders.csv – Transactional order data
- users.json – User master data
- restaurants.sql – Restaurant master data

## Tools & Technologies
- Python
- Pandas
- SQLite
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Processing
- Loaded data from multiple sources
- Performed LEFT JOINs to retain all orders
- Cleaned and prepared final dataset
- Exported final dataset as CSV

## Analysis Performed
- City-wise and cuisine-wise revenue analysis
- Gold vs Regular membership impact
- Restaurant rating vs revenue
- User spending behavior
- Seasonal revenue trends

##  Output
- `final_food_delivery_dataset.csv`
- `food_delivery_analysis.ipynb`

##  Conclusion
This project demonstrates an end-to-end data analytics workflow including data integration, cleaning, analysis, and visualization.
